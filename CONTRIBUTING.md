# Contributing to This Repository

Thank you for participating in the MNSUG GitHub Onboarding Workshop!

## How Members Contribute

1. **Fork** this repository by clicking the **Fork** button at the top-right of the page
2. Navigate to the section folder that applies to you (e.g. `01-account-creation/`)
3. Open the `README.md` inside that folder
4. Click the **pencil icon** to edit the file
5. Fill in your name, GitHub username, and all the required fields
6. Scroll down and commit with a clear message, e.g.:
   ```
   docs: add [Your Name] account creation notes
   ```
7. Open a **Pull Request** back to the `main` branch of this repository
8. Add a short description of what you added

## Pull Request Title Format

```
docs: add [Your Name] - Section [Number] notes
```

Example:
```
docs: add Oyimafu Emmanuel - Section 1 account creation notes
```

## Guidelines

- Fill in **your own** answers — do not copy from others
- Keep all personal secrets, passwords, or tokens **out** of your submissions
- Be respectful and professional in all content
- If you are unsure about something, add a question in your notes

## Questions?

Ask your facilitator or open a GitHub Issue on this repository.

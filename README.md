# MNSUG GitHub Onboarding Workshop

**Microsoft-Naija-Security-Usergroup (MNSUG) · June 2026**

Welcome to the official GitHub onboarding workshop repository for the Microsoft Naija Security User Group. This repo contains step-by-step guides covering everything from account creation to security best practices.

---

## How to Contribute as a Member

> **Members fork this repo, edit the Markdown files to add their content, then open a Pull Request.**

### Steps

1. **Fork** this repository — click the **Fork** button at the top-right of this page
2. **Edit** the relevant Markdown file inside the section folder that applies to you
3. **Add your content** — notes, screenshots descriptions, personal tips, or answers to lab exercises
4. **Commit** your changes with a clear message, e.g. `docs: add my account creation notes`
5. **Open a Pull Request** back to this repository — the facilitator will review and merge

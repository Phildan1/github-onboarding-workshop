# Security Policy

## Supported Versions

| Branch | Security Updates |
|--------|-----------------|
| `main` | Yes — actively maintained |

## Reporting a Vulnerability

Do **not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities via GitHub's private security advisory on this repository, or contact the MNSUG facilitator directly.

### Response Timeline

| Stage | Target |
|-------|--------|
| Acknowledgement | Within 48 hours |
| Assessment | Within 7 days |
| Resolution | Within 90 days |
